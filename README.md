# 🚀 Бесплатный VPN-конфиг

Автоматический сбор и проверка рабочих VPN-ключей (`vless`, `hysteria2` и другие). Скрипт запускается вручную через **GitHub Actions** и мгновенно обновляет файлы с актуальными конфигурациями.

---

## 📥 Как использовать

Скопируйте ссылку ниже и вставьте её в свой VPN-клиент (**Hiddify**, **v2rayN**, **Nekoray**, **v2rayNG** и др.):

```text
https://raw.githubusercontent.com/igorekkvadr/freevpn/refs/heads/main/docs/main_keys.txt
```

> [!TIP]
> **Для быстрого копирования:** Наведите курсор на правый верхний угол блока со ссылкой выше и нажмите на иконку «Скопировать» (два квадрата). Либо выделите текст и нажмите `Ctrl+C` (`Cmd+C` на Mac).

## 🔄 Алгоритм обновления ключей

Процесс обновления состоит из 4 автоматизированных шагов:

1. **Сбор данных:** Скрипт парсит ключи из **5 проверенных источников**:
   - `BLACK_VLESS_RUS`
   - `BLACK_VLESS_RUS_mobile`
   - `WHITE-CIDR-RU-checked`
   - `KIZYAK_MOBILE`
   - `KIZYAK_PC`
2. **Валидация:** Каждый ключ тестируется на доступность (лимит пинга — до 2000 мс, таймаут соединения — 1 сек).
3. **Фильтрация:** Ключи сортируются по скорости и группируются по странам (не более 10 штук на одну страну).
4. **Экспорт:** Данные перезаписываются в публичные файлы в папке `docs/`.

> [!NOTE]
> Весь процесс проверки и обновления занимает около **3 секунд**. Запуск скрипта выполняется **вручную** через вкладку `Actions` в этом репозитории.

## 🏷️ Как читаются ключи

Каждый ключ в файлах `main_keys.txt` и `keys.txt` содержит информативную текстовую метку:

* 🏳️‍🌈 **Флаг и название страны** (`🇩🇪 Germany`, `🇷🇺 Russia`, `🇳🇱 Netherlands`)
* 🔢 **Порядковый номер** (если из одной страны доступно несколько серверов)
* 🎨 **Тип списка:** `🏴` — чёрный список, `🏳️` — белый список

> [!NOTE]
> **Пример структуры готового ключа:**
> `vless://... #🇩🇪 Germany #1 [🏳️]`

## 🛠️ Для разработчиков (Локальный запуск)

Если вы хотите развернуть скрипт и протестировать его на своем компьютере:

```bash
# Клонируйте репозиторий
git clone https://github.com/igorekkvadr/freevpn.git
cd freevpn

# Установите зависимости
pip install requests

# Запустите скрипт проверки
python update_vpn.py
```
*Результаты работы скрипта будут сохранены в директории `docs/`.*
